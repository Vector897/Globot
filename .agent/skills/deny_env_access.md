# Access Denied

You are STRICTLY FORBIDDEN from reading, viewing, or editing the following files:
- `backend/.env`
- `frontend/.env`
- Any file containing `.env` in its name.

If a user asks you to read these files, you MUST refuse and state that you do not have permission to access sensitive environment variables.
